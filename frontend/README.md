# 📖 BookHeaven - Frontend (React + Vite)

Esta es la aplicación cliente para **BookHeaven**, construida con **React 19** y **Vite 7**. Proporciona una interfaz moderna, premium y dinámica para el consumo de libros digitales.

---

## 🚀 Tecnologías Principales
- **Framework:** React 19.x (Hooks & Context API)
- **Build Tool:** Vite 7.x
- **Gráficos:** Recharts (Analíticas para Admin)
- **Cliente API:** Axios
- **Navegación:** React Router Dom v7

## ✨ Funcionalidades Destacadas
- **UI Premium:** Diseños con `Vanilla CSS`, micro-interacciones y transiciones dinámicas.
- **Admin Dashboard:** Visualización interactiva de analíticas demográficas de lectura.
- **Auth Flow:** Registro detallado de usuarios con preferencias y validaciones en tiempo real.
- **Lector Integrado:** Visor de PDF y Reproductor de Audio (Estructura inicial activa).
- **Favorites & Reviews:** Gestión de lista de lectura y feedback social.

## 🛠️ Comandos Rápidos
```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Compilar para producción
npm run build

# Limpiar y Previsualizar
npm run preview
```

## 🏗️ Estructura de carpetas
- `src/api`: Gestión de llamadas axios.
- `src/components`: Componentes reutilizables.
- `src/context`: Manejo del estado global (Auth, etc).
- `src/pages`: Vistas completas de la aplicación.
- `src/styles`: Estilos compartidos y específicos por componente.
- `src/utils`: Funciones de utilidad (Formateo de fechas, validaciones).
